# Portfolio Optimization using Pyomo and IPOPT

This project performs portfolio optimization using real stock data from Yahoo Finance. 
It calculates monthly returns and plots the efficient frontier using the IPOPT solver.

## Features
- Automatic stock data download via `yfinance`
- Monthly return calculation
- Pyomo-based portfolio optimization
- Efficient frontier and allocation plots
